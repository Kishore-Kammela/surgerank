# Glossary

## Purpose
Define shared terms and abbreviations used across the scope package.

## Terms

| Term | Meaning |
| --- | --- |
| Agency | A customer account that manages multiple client workspaces |
| Workspace | A client-specific unit inside an agency account |
| GEO | Generative Engine Optimization; visibility in AI-generated answers |
| GSC | Google Search Console |
| P0/P1/P2 | Priority levels (must-have / should-have / deferred) |
| Activation | Completion of required onboarding and first-value actions |
| Weekly value-active workspace | Workspace with fresh data and active reporting usage |
| Launch gate | Metric threshold required before advancing launch stage |
| RLS | Row-level security policy enforcement in database |
| ARPA | Average revenue per agency account |
