# V1 Roadmap

## Purpose
Provide a phased roadmap with milestones, dependencies, and outcomes.

## Executive Summary
- Target: launch-ready V1 in 20 weeks with a design-partner cohort.
- Sequence: foundation -> core data value -> reporting -> hardening -> launch.

## Phases
1. Foundations (weeks 1-2)
2. Integrations and dashboards (weeks 3-5)
3. SEO audits and quality baseline (weeks 6-8)
4. Keywords and competitor basics (weeks 9-11)
5. Limited GEO and reporting (weeks 12-16)
6. Hardening and launch readiness (weeks 17-20)

## Dependencies
- Supabase schema and RLS before core module development.
- CI/CD baseline before team-scale feature work.
- KPI instrumentation before partner expansion.

## Review Cadence
- Weekly planning review.
- Weekly launch-gate scorecard.
