# Demo Narrative Template

## Purpose
Guide consistent, high-clarity demos for mixed technical and non-technical audiences.

## Demo Objective
- What decision or confidence outcome should this demo create?

## Audience
- Who is attending?
- What each audience group needs to understand?

## Narrative Flow
1. Problem context
2. What changed since last demo
3. Live walkthrough of critical workflow
4. Value demonstrated
5. Risks or limitations
6. Next actions and asks

## Demo Checklist
- [ ] Data and scenarios prepared
- [ ] Success and failure paths shown
- [ ] Metrics impact explained
- [ ] Decisions/feedback captured
