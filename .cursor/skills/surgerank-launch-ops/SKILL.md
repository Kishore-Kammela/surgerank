# SurgeRank Launch Ops

## Goal
Move from build to paying customers with clear operational gates.

## Launch Model
- Design-partner first (5-8 agencies)
- Weekly feedback cadence
- Controlled public launch only after gate metrics pass

## Activation Definition
A workspace is activated when all are complete:
1. GSC connected
2. First crawl complete
3. At least 50 keywords imported
4. GEO query set configured
5. First report exported or scheduled

## Core Operating Cadence
- Weekly product/ops review
- Weekly partner check-ins
- Weekly KPI dashboard review
- Biweekly scope freeze validation

## Launch Gates
- Activation rate >= 70% by day 10
- Week-4 retained usage >= 60%
- Data pipeline success >= 98%
- Gross margin projection >= 70%
- Minimum paying agencies threshold reached

## Deliverables
- Partner onboarding runbook
- Weekly scorecard
- Risk register
- Launch go/no-go report
